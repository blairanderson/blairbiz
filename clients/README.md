# Clients

Private directory — excluded from Jekyll build, never published to the site.

One folder per client. Each folder contains:
- `profile.md` — business overview, contact info, context
- `YYYY-MM-DD.md` — individual meeting notes (one file per meeting)

## Quick Start

Copy `_template/meeting.md` for each new meeting.
Copy `_template/profile.md` when onboarding a new client.
